ScratchFirst GameMr.Brunner's Fruit Adventure
The purpose of this game is to shoot all the unhealthy foods
and to get the highest score you can. Developing this game started
with deciding the objective of the game and how the player would could
the sprites. A specific script that I focused on was the script that
allowed the sprite to move using the left and right arrow keys and pressing
the up arrow key to launch the projectile upwards. Another distinct script
that I focused on was the score-keeping script. This script was especially
important because I had to think of how to increase the difficulty as the
player got a higher score. Overall the difficulty of making this game
included debugging the game so that it worked properly, however this was
resolved by finding any illogical gaps in the script.
